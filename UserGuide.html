MonsterCalc User Guide

MonsterCalc is a live scratchpad calculator for technical work. Type one expression per line in the left pane and the result appears on the matching line in the right pane.

How To Use MonsterCalc

  • Type one expression per line. Blank lines are allowed.
  • Results update live as you type.
  • Use ans to reference the most recent successful result.
  • Use line1, line2, and so on to reference successful results by line number.
  • Click a line number in the editor gutter to insert the matching lineX token at the current cursor position.
  • Use File -> Load Recent… to reopen one of the last autosaved sheets, sorted by last edit time with a preview of the first line.
  • Use the top-right menus for quick insertion of math, symbols, units, and programming helpers.
  • Autocomplete appears inline after a unique three-character match for built-in functions.
  • Hover an integer result in the right pane to see the same value in default, decimal, hex, and binary formats.

Tip: comments start with #. Example: vdiv(5, 10k, 10k) # voltage divider

Practical Tips

  • Use the result format setting if you prefer scientific, engineering, or SI-style output.
  • If a line is incomplete or invalid while typing, the result stays blank until the expression is valid.
  • The results pane is read-only, but you can copy from it with Ctrl+C.
  • Starting a new sheet with the + button autosaves the current sheet only when it contains content, and those snapshots appear in File -> Load Recent….
  • Integer hover previews are useful for programming work because they keep the normal display compact while exposing decimal, hex, and binary on demand.
  • Clicking a line number inserts a lineX reference without moving your current edit position.
  • Autocomplete hints appear in the editor for built-in functions and their expected arguments.
